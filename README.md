# 🐍 Ciência de Dados DIO - iFood

Repositório criado para armazenar os desafios e projetos desenvolvidos durante o **Bootcamp de Ciência de Dados da DIO em parceria com o iFood**.

Os códigos presentes neste projeto representam a prática dos principais conceitos de programação e análise de dados utilizando Python.

---

## 📚 Conteúdos abordados

Durante o desenvolvimento das atividades foram trabalhados conceitos como:

- ✅ Lógica de programação com Python
- ✅ Estruturas condicionais e de repetição
- ✅ Manipulação de dados
- ✅ Resolução de desafios práticos
- ✅ Análise exploratória de informações
- ✅ Uso de Jupyter Notebook para experimentação

---

## 📂 Arquivos do projeto
